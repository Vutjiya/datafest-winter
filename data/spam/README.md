# Spam Dataset

**Source**: [Spam Dataset](https://archive.ics.uci.edu/dataset/94/spambase)

**Description**: This dataset contains email features for classifying emails as spam or non-spam, with $4601$ total observations and $58$ variables. The dataset was created by Mark Hopkins, Erik Reeber, George Forman, and Jaap Suermondt at Hewlett-Packard Labs and donated in $1999$. The task is to determine whether a given email is spam or not spam.
