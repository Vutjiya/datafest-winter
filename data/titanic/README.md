# Titanic Dataset

**Source**: [Titanic Dataset](https://www.kaggle.com/competitions/titanic/data)

**Description**: This dataset contains passenger information from the RMS Titanic, with $891$ total observations and $12$ variables. The task is to predict whether a passenger survived or did not survive the Titanic disaster based on features such as age, sex, passenger class, and family relationships.

**Variables**:

| Variable Name | Type        | Description                                                          |
| ------------- | ----------- | -------------------------------------------------------------------- |
| `id`          | Categorical | Unique identifier for each passenger                                 |
| `survived`    | Categorical | Survival status (0 = did not survive, 1 = survived)                  |
| `pclass`      | Categorical | Passenger class (1 = 1st class, 2 = 2nd class, 3 = 3rd class)        |
| `name`        | Categorical | Passenger name                                                       |
| `sex`         | Categorical | Passenger sex (male, female)                                         |
| `age`         | Continuous  | Passenger age in years                                               |
| `sibsp`       | Continuous  | Number of siblings or spouses aboard the Titanic                     |
| `parch`       | Continuous  | Number of parents or children aboard the Titanic                     |
| `ticket`      | Categorical | Ticket number                                                        |
| `fare`        | Continuous  | Passenger fare in British pounds                                     |
| `cabin`       | Categorical | Cabin number                                                         |
| `embarked`    | Categorical | Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton) |
